## Инструкция по установке:
#### Клонировать репозиторий
```sh
git clone https://github.com/giftfromheaven/forkify_app.git
```
#### Установить все необходимые для работы модули
```sh
npm install
```
#### Запуск проекта
```sh
npm run start
```
